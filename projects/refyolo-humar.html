RefYOLO-HuMaR

A unified framework for multi-human referring expression grounding that integrates token-level text-visual alignment with YOLO-based detection for simultaneous localization and identification of multiple individuals from natural language descriptions.

43.23% MSR 95K+ Images 540K+ Annotations Ongoing Research

Problem Statement

Existing referring expression comprehension (REC) methods predominantly focus on single-target grounding — localizing one individual per query. In real-world scenarios, natural language descriptions often refer to multiple people simultaneously (e.g., "the two people shaking hands near the podium"). Current systems fail to handle such multi-target references due to architectural limitations in both the language understanding and visual grounding pipelines.

Motivation

Multi-human referring expression grounding is critical for applications in surveillance, human-robot interaction, and assistive technologies. The ability to ground multiple individuals from a single query enables more natural and efficient human-machine communication. This project addresses the gap between single-target REC methods and the practical need for multi-target visual grounding.

Dataset

A large-scale dataset was constructed by integrating annotations from four major sources:

  • CrowdPose — Dense multi-person pose estimation scenes
  • MPII Human Pose — Diverse everyday human activities
  • OpenImagesV7 — Web-scale images with rich annotations
  • MS COCO — Complex scenes with multiple interacting people

The combined dataset contains 95K+ images with 540K+ multi-human referring annotations, making it one of the largest resources for this task.

Methodology

The framework builds on the YOLO detection backbone, extended with a language-aware grounding head. Key design choices include:

  • MiniLM-based sentence encoding for efficient query representation
  • Token-level text-visual alignment using bidirectional cross-attention
  • Multi-Instance Ranking Loss (MIRL) for hard negative mining
  • Joint training on detection and grounding objectives

Architecture Overview

RefYOLO-HuMaR Architecture Diagram — Visual backbone, Language encoder, Cross-attention fusion, Multi-target grounding head

Visual Backbone

The visual branch uses a YOLO-based feature pyramid network to extract multi-scale spatial features from the input image, providing both global scene context and fine-grained local details for accurate person localization.

Language Encoder

Natural language queries are processed through a MiniLM encoder to produce token-level embeddings. These embeddings preserve fine-grained linguistic structure necessary for distinguishing between multiple referred individuals.

Cross-Attention Fusion

Bidirectional cross-attention modules align visual region features with language token embeddings, enabling the model to jointly attend to relevant spatial regions and linguistic cues for each referred person.

Experiments

Training is conducted on the constructed multi-human dataset with standard train/val/test splits. The model is evaluated against single-target baselines adapted for multi-target grounding, measuring Mean Success Rate (MSR) and per-instance localization accuracy.

Results

Preliminary results demonstrate 43.23% Mean Success Rate on the validation set, establishing a competitive baseline for the multi-human REC task. The token-level alignment approach shows particular strength in disambiguating individuals with similar visual appearances.

Qualitative Results — Grounding visualizations on test images

Failure Cases

Current failure modes include heavy occlusion scenarios where multiple individuals overlap significantly, and ambiguous language descriptions that can refer to variable numbers of people. Edge cases with extreme crowd density also present challenges for the grounding head.

Ablation Study

Ablation studies are in progress to evaluate the contribution of each component: cross-attention depth, MiniLM vs. alternative encoders, MIRL loss weighting, and multi-scale feature fusion strategies.
