CIPS-Net

A vision-language segmentation framework for compositional, instruction-driven segmentation of overlapping pathology classes in histopathology images, leveraging a compositional graph reasoning module for instruction-aware dense decoding.

0.795 Macro Dice Completed
Code (Coming Soon) Paper (In Preparation)

Problem Statement

Histopathology image analysis requires identifying and segmenting multiple overlapping tissue structures and pathological features. Traditional segmentation approaches use fixed-label prediction, treating each class independently. This paradigm fails when pathology categories overlap spatially, and it cannot generalize to new tissue types without retraining. There is a need for instruction-driven segmentation that can selectively segment specific structures based on natural language prompts.

Motivation

Pathologists routinely analyze tissue specimens by focusing on specific morphological features described in clinical guidelines. An instruction-conditioned model that segments tissue structures based on textual descriptions would align more naturally with clinical workflows. This approach also enables a single model to handle multiple segmentation tasks, reducing the need for task-specific architectures.

Dataset

The framework is evaluated on standard histopathology segmentation benchmarks containing diverse tissue types with overlapping pathology annotations. The datasets include pixel-level masks for multiple pathological categories that frequently co-occur in the same regions.

Dataset Samples — Histopathology images with overlapping pathology annotations

Methodology

CIPS-Net introduces compositional graph reasoning into the segmentation pipeline. Key contributions include:

  • Instruction encoding using language models for conditional segmentation
  • Compositional graph module that captures inter-class relationships
  • Dense decoder with instruction-aware feature modulation
  • U-Net backbone enhanced with cross-modal attention layers

Architecture Overview

CIPS-Net Architecture — Encoder, Compositional Graph Module, Instruction-Conditioned Decoder

Instruction Encoder

Textual instructions describing the target pathology are encoded into a fixed-dimensional embedding vector. This embedding conditions the decoder to selectively activate features relevant to the queried pathology class.

Compositional Graph Module

A graph neural network module captures spatial and semantic relationships between different tissue structures. Nodes represent pathology classes while edges encode co-occurrence and spatial adjacency patterns learned from the training data.

Conditioned Decoder

The decoder receives multi-scale visual features from the encoder, modulated by the instruction embedding through FiLM (Feature-wise Linear Modulation) layers. This enables class-specific feature enhancement while suppressing irrelevant activations.

Experiments

The model is trained with standard cross-entropy and Dice loss functions, optimized using AdamW with cosine annealing learning rate scheduling. Experiments compare CIPS-Net against fixed-label U-Net baselines and other instruction-conditioned segmentation approaches.

Results

CIPS-Net achieves a Macro Dice score of 0.795, outperforming fixed-label U-Net baselines on overlapping pathology segmentation. The instruction-conditioned approach shows improved performance on rare pathology classes due to the compositional reasoning capability.

Segmentation Results — Comparison with baselines across pathology classes

Failure Cases

The model struggles with extremely rare pathology classes that have fewer than 50 annotated instances in the training set. Additionally, highly ambiguous instructions that could refer to multiple similar tissue types lead to degraded segmentation boundaries.

Failure Case Analysis — Rare classes and ambiguous instructions

Ablation Study

Component-wise ablation demonstrates the contribution of each module: the compositional graph module provides +3.2% Dice improvement, while instruction conditioning adds +5.1% over the vanilla U-Net baseline.

Future Work

  • Evaluation on additional histopathology benchmarks (CAMELYON, PanNuke)
  • Semi-supervised learning to leverage unlabeled pathology slides
