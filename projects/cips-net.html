CIPS-Net

Cross-Layer Instance-Prototype Similarity Based Feature Aggregation for Multi-Task Dense Perception. A novel approach to dynamically weight multi-scale features for autonomous driving tasks.

Problem Statement

Multi-task dense perception for autonomous driving requires simultaneous prediction of semantic segmentation, depth estimation, and surface normal maps. Existing multi-task architectures often use static feature aggregation that fails to account for task-specific feature requirements at different scales. This leads to suboptimal performance when tasks have conflicting gradient signals or demand features from different abstraction levels.

Motivation

Autonomous vehicles need to understand their environment comprehensively and efficiently. A unified model that performs multiple perception tasks reduces computational overhead and ensures consistent scene understanding. The key challenge is designing feature aggregation mechanisms that adaptively weight contributions from different network layers based on task requirements and instance characteristics.

Methodology

CIPS-Net introduces a novel feature aggregation mechanism based on instance-prototype similarity. Key contributions include:

  • Cross-Layer Feature Aggregation: Dynamic weighting of multi-scale features based on learned prototypes
  • Instance-Prototype Similarity: Per-pixel similarity computation that adapts feature fusion to local context
  • Task-Specific Decoders: Lightweight task heads that leverage aggregated features for final predictions
  • Gradient-Based Task Balancing: Adaptive loss weighting to handle conflicting gradients across tasks

Architecture Overview

CIPS-Net Architecture — Shared Encoder, Instance-Prototype Module, Task-Specific Decoders

Feature Encoder

A shared ResNet-based encoder extracts multi-scale feature maps from input images. Features from multiple layers are passed to the cross-layer aggregation module.

Instance-Prototype Module

Learnable prototype vectors represent canonical feature patterns for each task. Per-pixel similarity scores are computed between local features and prototypes, producing adaptive weights for feature aggregation.

Task Decoders

Lightweight convolutional decoders process aggregated features to produce final predictions for semantic segmentation, depth estimation, and surface normals.

Tasks

  • Semantic Segmentation: Pixel-level classification of road elements
  • Depth Estimation: Monocular depth prediction from single images
  • Surface Normal Estimation: Per-pixel surface orientation prediction

Datasets

The framework is evaluated on standard autonomous driving benchmarks:

  • NYUv2: Indoor scene understanding with dense annotations
  • Cityscapes: Urban driving scenes with semantic labels
  • PASCAL-Context: General scene parsing benchmark

Key Contributions

  • Novel instance-prototype similarity mechanism for adaptive feature aggregation
  • Competitive multi-task performance while maintaining parameter efficiency
  • Reduced negative task interference through dynamic feature routing
