Overview

Doc2Data is an AI-powered document digitization system developed for the Andhra Pradesh Police Department. The system automates the extraction of structured information from unstructured handwritten and printed documents, including FIRs, statements, and administrative records.

Key Features

  • Handwriting Recognition: Custom OCR pipeline optimized for Telugu and English handwritten text
  • Information Extraction: Named Entity Recognition (NER) and key-value extraction from unstructured text
  • Document Classification: Automatic classification of document types (FIR, statement, etc.)
  • Structured Output: Conversion of extracted information into queryable database format
  • Quality Assurance: Confidence scoring and human-in-the-loop verification workflow

Technical Approach

The system employs a multi-stage pipeline:

  • Stage 1: Document preprocessing and image enhancement for OCR
  • Stage 2: Text extraction using custom-trained OCR models
  • Stage 3: NLP-based information extraction using fine-tuned transformers
  • Stage 4: Rule-based post-processing and validation
  • Stage 5: Structured data generation and database integration

Impact

The Doc2Data system aims to significantly reduce manual data entry workload for police administrative staff, improve data accuracy, and enable faster retrieval and analysis of historical records.
